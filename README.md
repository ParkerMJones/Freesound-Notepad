# Freesound-Notepad

## Purpose

* Pulls a random set of sounds from Freesound.org and maps them to keyboard input

## Built With

* HTML
* CSS
* JS
* Skeleton CSS (framework)
* Freesound (API)
* Pexel Images (API)
